História:

Você é tripulante de um navio mercante, passando por um pequeno conjunto de ilhas, quando uma tempestade te pega e após travar horas de combate com a natureza um raio terrível atinge o seu navio! Mesmo chovendo o fogo se espalhou por todo o navio permitindo que a água entrasse pelo convés, felizmente você conseguiu guiar o navio até uma pequena ilha antes que ele naufragasse, porém, ele ficou encalhado na costa. O fogo parou e o sol apareceu - mas está começando a se pôr...


Você e sua equipe devem sobreviver na ilha até que outro navio mercante passe, mas quem sabe quanto tempo isso vai demorar. Enquanto isso você pode explorar a ilha, talvez haja vestigios de outros navios que tiveram problemas semelhantes na ilha.

Este mapa deve ser jogado no multiplayer, embora você possa tentar jogar sozinho. De qualquer forma poste nos comentários sua opinião e os videos que gravar para mostrar-nos como se divertiu!

Como instalar neste mundo:

Para jogar no mundo primeiro extraia o arquivo usando um programa como o WinRAR ou 7Zip , você pode descompactar o arquivo em qualquer lugar agora.
Isto deve criar uma pasta chamada "Survival Island - Stranded", que deve ser movida para a pasta .minecraft/saves (%appdata%/.minecraft/saves).

Se você não sabe como encontrar a pasta do jogo siga estes passos (para Windows):
1. clique no botão "Iniciar" no canto inferior esquerda da tela.
2. Na caixa de pesquisa digite %appdata% e clique na pasta "roaming".
3. Na "roaming" você deve encontrar a pasta .minecraft.
4. Aqui você vai encontrar uma pasta chamada "savas", que deve ter todos os seus mundos lá, mova a pasta "Survival Island - Stranded" para cá.

Objetivos:

# 1. Plantar 10 árvores
# 2. Conquistar um calabouço
# 3. Faça um bloco de lã usando linhas (string)
# 4. Faça todos os tipos de lã coloridas! (Se houver uma lã difícil de obter, favor postar nos comentários)
# 5. Descobrir um segredo do mapa
# 6. Faça uma fazenda de trigo que produza mais de 32 itens
# 7. Use uma abóbora para fazer um espantalho no meio de sua fazenda de trigo
# 8. Faça uma fazenda de melância e abóbora
# 9. Faça uma fazenda de cana-de-açúcar automática

Extra para os profissionais:

# 10. Junte a terra entre as 3 ilhas para fazer uma grande ilha!
# 11. Construa um castelo (realizado por BullyGamers)
# 12. Coloque uma abóbora na cabeça
# 13. Reconstrua o navio, não importa como desde que se pareça com um barco. (Se você tiver Movecraft ou qualquer mod que possa mover o navio você poderá usar isso para navegar para uma aldeia)
# 14. Obter uma armadura de ouro completa e fazer um trono para sentar-se sobre ele com orgulho!
# 15. Faça uma biblioteca com muitas estantes
# 16. Criar uma pequena vila ao redor do castelo que você construiu
# 17. Faça um trampolim até o nível das nuvens, salte e faça uma volta de 720° (fazer duas voltas completas), antes de chegar a água
# 18. Encontre 5 melões sobre a água!
# 19. Construa um iglu
# 20. Construa uma armadilha de mob uma dungeon.
# 21. Construa um gerador de pedras (sugestão popular)
# 22. Fazer um Golem (de ferro ou boneco de neve, se conseguir)
# 23. Derrote o Enderdragon
